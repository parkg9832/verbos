# Verbos CRM Dashboard

Next.js 기반 사내 업무 자동화 대시보드 MVP입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`으로 확인합니다.

## 빌드

```bash
npm run build
```

## GitHub Pages

`main` 브랜치에 푸시하면 GitHub Actions가 정적 사이트를 빌드해 GitHub Pages로 배포합니다.
